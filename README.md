# Urban Growth Engines

A comprehensive webpage exploring development economics with a focus on how major cities drive development diffusion and poverty reduction in emerging economies.

## Overview

This project provides an informative resource on:

- **Development Diffusion**: How economic progress spreads from urban cores to peripheral regions
- **Urban Growth Engines**: The role of major cities in catalyzing economic transformation
- **Poverty Reduction**: Mechanisms through which urbanization reduces poverty in developing nations
- **Case Studies**: Examples from Shenzhen, Bangalore, Lagos, and Nairobi

## Features

- Modern, responsive design optimized for all devices
- Interactive data visualizations
- Smooth scroll animations
- Accessible navigation
- Mobile-friendly interface

## Topics Covered

1. **The Urban Imperative** - Why cities matter for development
2. **Mechanisms of Diffusion** - How development spreads:
   - Agglomeration economies
   - Infrastructure investment
   - Human capital formation
   - Market access
   - Institutional development
   - Remittances & capital flows
3. **Urban Growth Engines in Action** - City case studies
4. **The Poverty Reduction Channel** - Direct and indirect effects
